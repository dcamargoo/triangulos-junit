# Triângulos - Testes com JUnit

Projeto de Teste de Software utilizando JUnit para validar se três valores podem formar um triângulo e identificar seu tipo.

## Objetivo

O projeto verifica:

- Se três valores podem formar um triângulo
- Se o triângulo é equilátero (3 lados iguais)
- Se o triângulo é isósceles (2 lados iguais)
- Se o triângulo é escaleno (3 lados diferentes)

## Tecnologias utilizadas

- **Java 21**
- **JUnit 4**
